# TikTok 虚拟产品 — 第一周执行清单

## 今天：搭基础设施

### 1. 注册 Payhip（免费）
- 打开 https://payhip.com 注册
- 创建产品 → 选「Digital Product」
- 上传你的第一个产品文件（先做个简单 PDF 测试）
- 定价 → 获取购买链接
- **把链接替换到 `index.html` 里每个产品的 `onclick` 和底部按钮**

### 2. 部署落地页（免费，5 分钟）
```bash
# 方法 A：GitHub Pages（推荐）
# 1. 在 GitHub 新建仓库 tiktok-store
# 2. 把 index.html 推上去
# 3. Settings → Pages → 选 main 分支 → Save
# 4. 拿到你的链接：https://你的用户名.github.io/tiktok-store/

# 方法 B：Netlify Drop
# 打开 https://app.netlify.com/drop
# 直接把 index.html 拖进去 → 即时上线
```

### 3. TikTok 账号设置
- 个人简介里放落地页链接（Link in Bio）
- 头像用统一视觉（你的剪映作品截图）
- 账号名建议：`@kyrastudio` 或 `@kyra.template`
